Title: Custom Panel file previews
----

Subtitle: No more blind dates with your files

----

Teaser:

Say goodbye to mystery files! With Kirby's new custom Panel file previews, you can set up the Panel to preview your custom file formats. Whether it's a crucial document or 3D animation, your editor will know exactly what they're dealing with.

Kirby 5 also ships with default basic previews for video and audio files.

----

Class:

```php
class PdfFilePreview extends FilePreview
{
    public function __construct(
        public File $file,
        public string $component = 'k-pdf-file-preview'
    ) {
    }

    public static function accepts(File $file): bool
    {
        return $file->extension() === 'pdf';
    }
}
```
